
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build a classic Hangman game in Python that uses strings, loops, conditionals, and user input to let players guess a secret word before they run out of attempts.

## 📝 Tasks

### 🛠️ Build the Hangman Game

#### Description
Create a Python program that randomly selects a secret word and lets the player guess letters until they find the word or use all attempts.

#### Requirements
Completed program should:

- Randomly select a word from a predefined list.
- Accept letter guesses from the player.
- Show the current word progress using blanks and revealed letters (e.g. `_ a _ g _ a _`).
- Track the remaining incorrect guesses.
- End the game when the player guesses the word or runs out of attempts.
- Display a win message if the player succeeds and a lose message if they fail.
